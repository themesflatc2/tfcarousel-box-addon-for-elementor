== Description ==

TFCarousel Box Addon For Elementor requires Elementor to be installed and activated. This plugin TFCarousel Box Addon For Elementor you can put everything in a carousel slide. Just design the template in Elementor, Then create a slide and include template.

== Frequently Asked Questions ==

= Can I use the plugin without Elementor Page Builder? =

No. You cannot use without Elementor since it's an addon for Elementor.

= Does it work with any theme? =

Absolutely! It will work with any theme where Elementor works.
